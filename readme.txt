URL Screenshot tool for your website

== Description ==

The user takes the photo by entering the site address in the form that it is on its site. This plugin is using 3rd party api service.

you can use following api services:
- <a href="https://www.abstractapi.com/ip-geolocation-api">abstractapi.com</a> / <a href="https://www.abstractapi.com/legal/legal">Terms and Policies</a>
- new API Coming soon

Features:
- [tida_screenshot] shortcode for showing URL Screenshot form
- Page Builder Blocks look like Gutenberg : Coming Soon

== Frequently Asked Questions ==

= How I use this plugin =

You can add shortcode [tida_screenshot] in anywhere of your website like page or page builders blocks.

= How I get a api key =

You must go to website that you select in api key service, then sign up and get a free api key. if you select ip-api api, you will don't need to get the api key.
